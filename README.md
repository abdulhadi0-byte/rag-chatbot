# 🤖 RAG Chatbot

A Retrieval Augmented Generation (RAG) Chatbot that lets you upload any text document and ask questions about it using AI.

## 🚀 Live Demo
[Click here to try the app](https://huggingface.co/spaces/abdulhadi0239209/rag-chatbot)

## 📌 What is RAG?
RAG stands for Retrieval Augmented Generation. Instead of relying on pre-trained knowledge alone, it searches your uploaded document for relevant information and returns accurate answers based on your content.

## ✨ Features
- Upload any `.txt` document
- Ask questions about your document
- Get instant AI-powered answers
- No API key required
- Clean and modern UI

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- HuggingFace Transformers
- FAISS Vector Store
- Sentence Transformers

## ⚙️ How to Run Locally

```bash
git clone https://github.com/abdulhadi0-byte/rag-chatbot.git
cd rag-chatbot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
